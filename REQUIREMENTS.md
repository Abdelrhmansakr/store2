# API endpoint Users

- POST /api/users

  `create new User`

- GET /api/users

  `GET all users`

- GET /api/users/:id

  `GET one user by id`


# API endpoint product

- POST /api/product

  `create new product`

- GET /api/product

  `GET all product`

- GET /api/product/:id

  `GET one product by id`


# API endpoint order

- POST /order

  `create new order`

- GET /orderu/:userId

  `GET order by id user`

- GET /orders/:ordersId

  `GET order by id orders`
